## Short Demo Clip Outline

1. Start at README and open `docs/portfolio-architecture.md`.
2. Run `make demo` and highlight run ID, trace ID, dashboard, report, and evaluation paths.
3. Start `make dev`.
4. Open `/app/admin` to show the platform configuration surface.
5. Open `/app/artifacts` to preview generated artifacts and lineage.
6. Open `/run-inspector/runs/{run_id}` to show execution graph state and events.
7. End on `/metrics` or Grafana to show operational signals.
