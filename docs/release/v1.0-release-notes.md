# v1.0 Release Notes Draft

Autonomous Enterprise AI OS v1.0 is a portfolio-ready release candidate for a durable multi-agent
workflow platform. It demonstrates how specialized AI agents can collaborate on an enterprise
analytics workflow while preserving state, lineage, policy decisions, and operational telemetry.

## What This Release Demonstrates

- Planner-generated workflow graphs with typed nodes, dependencies, expected artifacts, and risk.
- Durable run lifecycle with repository-backed runs, graph nodes, workflow jobs, artifacts, events,
  audit events, evaluations, and checkpoints.
- Specialized agents for planning, data retrieval, analytics/code, visualization, reporting,
  evaluation, security policy, and deployment approvals.
- Procurement analytics demo that produces schema, quality, KPI, code, chart, dashboard, report,
  evaluation, event, trace, and metrics artifacts.
- Browser control plane for runs, artifact previews, admin configuration, and run inspection.
- Security governance with tool permissions, approval gates, destructive-action blocks, audit events,
  and secret redaction.
- Observability through OpenTelemetry trace IDs, Prometheus metrics, Grafana dashboard definitions,
  optional MLflow, and optional LangSmith.
- Production-shaped operations with Docker Compose, Kubernetes overlays, runtime config validation,
  and an AWS deployment path using Terraform.

## Primary Demo

The canonical release demo is:

```bash
make install
make demo
make dev
```

Then open:

- `http://127.0.0.1:8000/app`
- `http://127.0.0.1:8000/app/artifacts`
- `http://127.0.0.1:8000/app/admin`
- `http://127.0.0.1:8000/run-inspector/runs/{run_id}`
- `http://127.0.0.1:8000/metrics`

## Out Of Scope For v1.0

- Multi-tenant enterprise SSO and billing.
- A general-purpose visual workflow builder.
- Managed production Snowflake credential rotation.
- Running arbitrary untrusted user code without additional sandboxing.
- Multi-cloud parity beyond the first AWS deployment path.

## Suggested GitHub Release Copy

Autonomous Enterprise AI OS v1.0 demonstrates a full multi-agent analytics operating system:
planner-generated execution graphs, durable orchestration, specialized agents, policy gates,
